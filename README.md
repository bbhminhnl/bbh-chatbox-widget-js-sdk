## Cài đặt

Sử dụng pnpm để cài đặt SDK:

```bash
pnpm i git+https://github.com/bbhminhnl/bbh-chatbox-widget-js-sdk-sdk.git
```

Sử dụng npm để cài đặt SDK:

```bash
npm i git+https://github.com/bbhminhnl/bbh-chatbox-widget-js-sdk-sdk.git
```
